Experimental topic

Algorithm Realization of Huffman Tree and Huffman Coding

Experimenta Purpose

  1. Understand the application examples of the tree, and be familiar with the construction method of the Huffman tree and the application of Huffman coding

  2. Understand the application process of Huffman tree in the field of communication and coding.

Experimental Requirements

  1. Input an English essay of 100-200 words and save it in a file a.

  2. Write a function to count the number n of letters appearing in the passage and the number of occurrences of each letter.

  3. Write a function that uses the number of occurrences of letters as the weight, build a Haffman tree (n leaves), and give the Haffman code of each letter.

  4. Encode the original short text with each letter code, and store the coded text in file b.

  5. Use the Haffman tree to decode the coded text in b, save the result in file c, and compare whether a and c are consistent, so as to check the correctness of encoding and decoding.

1. Save the entered text to a file

Each loop gets an input character and stores it in the file. Considering that there is a newline in the text of the input sample, the condition for exiting the loop is when the input is a newline and the next input is a space or a newline.

2. Get the weight of each letter

3. Obtain the corresponding Huffman code according to the weight

Select() is used to select two nodes with the smallest weight from nodes that currently have no parent class

HuffmanCoding() is used to encode characters by building a Huffman tree

4. Text encoding

Recode the articles stored in file a with Huffman code and store them in file b.

5. Text decoding

Re-decode the code stored in file b and store it in file c.

6. Compare the decoded file with the original file

Time complexity analysis:

Let the total number of words in the essay be m, and the number of coded characters be n

inputAndSave(), getWeight(), Compare() traverse the entire essay, the complexity is O(m);

The Select() function uses two for loops for n elements, and the complexity is O(n);

Encode(), and Decode() traverse the Huffman code for each character in the article, and the complexity is O(mn);

Experiment summary：

Note the case where there is only one character in the article. If there is only one character in the article, you can directly set the Huffman code of this character to 0.

When counting the number of occurrences of various letters in a short text, this program adopts the ASKII code of the letters as the order of the storage weight of the array number, which can quickly and accurately count the number of occurrences of each letter.

In addition, this program only encodes uppercase and lowercase letters according to the requirements of the topic. When dealing with characters other than letters (including spaces and carriage returns), the read-in characters are input into a new file as they are, so it can ensure that the format of the short text remains consistent before and after encoding and decoding.
